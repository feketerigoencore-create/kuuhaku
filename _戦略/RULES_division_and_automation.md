# 分担と自動化範囲のルール（AffiliateAuto 共通）

このファイルは、複数のClaude／環境（このチャット、Cowork等）で作業を分担する際の
共通ルールです。どの環境のClaudeも、このルールに従って動くこと。

====================================================
■ 役割分担
====================================================
【企画・下ごしらえ側】
- ジャンル・ニッチを受け取り、リサーチ→構成→下書きを作る。
- 成果物は下書きファイルまで（01_research / 02_outlines / 03_drafts）。

【投稿準備側】
- 受け取った下書きを、WordPressに載せられる形に整える。
- タイトル確定案、見出し構造の最終調整、リンク挿入位置のマーキング、
  公開前チェックの消し込み。
- 成果物は「あとは人間が貼って公開を押すだけ」のドラフト（WordPress下書き保存まで）。

【人間（オーナー）】
- ニッチの最終選定、構成のGO判断。
- 【要検証】箇所の裏取り（ストア名・商品名・品番・クーポン・還元率・セール時期・
  報酬単価・モザイク／未成年表現／AI申告の確認）。
- アフィリエイトリンクの設置。
- WordPressの公開ボタン。

====================================================
■ 自動化してよい範囲／禁止範囲
====================================================
【自動化OK】
- リサーチ、構成案、下書きの生成。
- 他ユーザー評価の傾向の要約（転載はしない）。
- 週次などの定期リサーチ（ファイル出力までで止める）。
- 下書きをWordPressに「下書き保存」する所まで。

【自動化禁止（必ず人間が手で行う）】
- WordPressへの公開（公開ボタン）。
- アフィリエイトリンクの設置。
- 【要検証】箇所を未確認のまま確定・反映すること。
- 上記を「承認なしで定期自動実行」する設定。

理由：公開とリンク設置は取り消せない操作。アダルト系は未成年表現・モザイク不備・
AI申告漏れでアカウント停止に直結するため、公開前に人間の目視確認を必ず挟む。
時短効果（数分）に対してリスク（アカウント・信用の喪失）が大きすぎる。

====================================================
■ 自動化のゴール（合意事項）
====================================================
「ワンクリックで公開できる状態まで」を上限とする。
= 下書き完成・タイトル確定・リンク位置マーク・公開前チェック済みのドラフトを
  WordPressに下書き保存。人間は中身を目視し、リンクを貼り、公開を押すだけ。
  → 公開前の関所（人間チェック）を必ず1つ残す。
