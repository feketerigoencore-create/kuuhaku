# Part 1. 一言でいうと（人に説明する用）

**広告枠を“安く仕入れて”、成果報酬の案件に流して“高く売る”。その差額（アービトラージ＝裁定取引）で稼ぐビジネス。**

## たとえ話
八百屋が「卸で安く仕入れて、店で高く売る」のと同じ。ただし扱うのは野菜ではなく**「広告のクリック（人の来訪）」**。

- **仕入れ**：広告ネットワークから、人の来訪（popなど）を1回0.03円くらいで大量に買う。
- **販売**：来た人を「成果報酬の案件」に流す。誰かが登録やインストールをすると、報酬が入る（例：1件 $1.94）。
- **利益**：`もらえる報酬 − 使った広告費` がプラスなら勝ち。

## お金の流れ（3者）
```
 [広告ネットワーク]        [自分＝メディアバイヤー]        [アフィリエイト網]

 （広告枠を売る）      お金を払って       （案件リンクを持つ）    成果が出たら
                     枠を買う                            報酬をくれる
        ↑___________________ 差額があなたの利益 ___________________↑
```

# Part 3. 具体的な1周（お金と人の動き）

1. **枠を買う**：HilltopAdsに入金し、「popを1万回表示」みたいに広告枠を買う。
2. **案件に流す**：買った来訪を、アフィリ網の案件リンク（例：ゲームのインストールページ）に飛ばす。
3. **成果が出る**：来た人の一部が「インストール」「メール登録」などをする＝**コンバージョン（成約）**。
4. **報酬が入る**：成約1件ごとにアフィリ網から報酬（例：$0.5〜$1.94）。
5. **差益を計算**：`報酬合計 − 広告費` がプラスなら利益。マイナスなら赤字。
6. **改善（ここが本番）**：どの広告枠（zone）が成約したかを見て、**成約しない枠を切り、成約する枠に寄せる**。これで赤字→黒字に近づける。

# Part 4. 最低限の用語集

- **pop / popunder**：サイトを開くと裏に“ポンッ”と開く広告。激安で大量に買える。
- **CPA / CPL / SOI / DOI / PPI**：成果の種類。SOI＝メール1個で成果（最軽量）／PPI＝インストールで成果／CPS＝購入で成果（重い）。
- **cold pop**：意図ゼロで飛ばされてきた人。だから**「1タップ or メール1個」など超軽い成果しか成約しない**。
- **zone（ゾーン）**：広告枠の単位。枠ごとに質がバラバラ。**利益は“勝つ枠だけ”に宿る**。
- **LP（プリランダー）**：案件の前に挟むワンクッションのページ（年齢確認など）。
- **中継ドメイン / チェーン**：`広告 → 中継 → 案件 → 最終着地` の一連の転送。**このどれか1つでも“汚れている”と審査で落ちる**。
- **ドメイン評判（VirusTotal / GSB）**：そのドメインがウイルス対策ソフトに「危険」と登録されていないか。**複数社が「危険」判定＝アウト**。
- **計測（トラッカー / postback / subid）**：どの枠が成約したかを記録する仕組み。これが無いと“丸腰”。

# Part 5. なぜ難しいのか（初心者がぶつかる4つの壁）

1. **入金の壁**：海外の広告ネットワークにドル入金する（決済でつまずく）。→ 今回は突破済み。
2. **審査の壁（最大）**：広告ネットワークは、`中継〜最終着地の全ドメイン評判` と `中身` を審査する。**チェーンに汚れが1つでもあると却下**。
3. **実績ゲートの壁**：良い案件・良い網ほど「実績のある人だけ」に絞る（新規お断り）。鶏と卵。
4. **成約＋計測の壁**：cold popは最軽量の成果しか成約しない。かつ**計測（どの枠が成約したか）が無いと改善できない**。

> プロと素人を分ける線は、**案件選びのセンスではなく「クリーンなチェーン × 計測 × 枠(zone)選別」**。

# Part 6. 僕らの全記録（時系列ダイジェスト）

**予算 約$50 で「広告開設→入金→配信→数字を見る→1回改善」を1周する学習が目的。利益より“仕組みの手触り”を得るのがゴール。**

1. **開設・入金**：Chaturbate(アフィリ)＋HilltopAds(広告主)登録、$100入金。海外決済という最大の関門を突破。
2. **8回の却下と原因究明**：
   - cam直・空LP → 「Unacceptable offer」（中身が薄い/露骨）
   - 出会い系 → 「Malicious domain」（MyLeadの転送ドメインが汚染）
   - **ゲームinstall → 初めて“承認・配信”**（着地＝Google Play＝クリーン）
3. **初配信の学び**：メキシコ×最小入札で1万表示＝約$4。だが**0成約**（成果が「install＋レベル10」と重すぎ＋bot＋オファーが途中で死亡）。
4. **プロの型を学ぶ**：計測(postback)なし・最小入札・枠選別なし・重い成果＝「確実に負ける組み合わせ」を全部やっていたと判明。
5. **計測(BeMob)構築 → 断念**：無料の共有ドメイン(bemobtrcks.com)が**GSBで「危険」判定**。独自ドメインは月$249。→ 不採用。
6. **LP・dating・gameで再挑戦**：
   - 却下理由が進化：「Invalid link（GEOロック着地）」「Empty targeting（フォーム不具合）」「Unacceptable offer」。
   - **フォームの罠**：カスタムのトグルは普通クリックだと内部データに反映されない（JSのネイティブclickで解決）。「URL invalid」表示の多くは非表示要素の誤検知だった。
7. **担当Mikeに確認（重要）**：アカBANではない。**ジャンル（ゲームもdatingも）はOK**。「Unacceptable」は毎回、**中継or最終着地のドメインがAV/GSBでフラグ**が主因。**出す前に全ドメインをVirusTotalで確認せよ**。
8. **原因を実測で確定**：MyLeadのゲーム中継 `sungoclick.space` が **1社→2社(複数AV)に悪化**。これが却下の正体。＝**MyLeadの共有中継ドメインが汚れた**という構造的問題。
9. **MyLeadに相談**：クリーンな中継への差し替えは無し。案内は「自前トラッカー(有料)」か「HideLink」。**HideLinkはクローキング（審査ボットを騙す）＝規約違反・BANリスク**なので不採用。
10. **別網 Cpamatica へ（②案）**：HilltopAds($51)はそのまま、**ドメインがクリーンな別網から案件を取る**方針。メッセンジャー必須の壁を**Telegram新規作成(@feketerigoencore)で突破**し、**登録完了→承認待ち(12〜72h)**。

# Part 7. 学びの核心（3行）

1. **ラクに稼げる魔法のボタンは無い。** 時間・お金・スキルのどれかは必ず払う。
2. **初心者の本当の壁は「入金」でも「英語」でもなく「審査（＝チェーン全ドメインの評判）」と「実績ゲート」。**
3. **利益は“クリーンなチェーン × 計測 × 勝つ枠への集中”で生まれる。** 案件のセンスは二の次。
