## 1. 結論：1周のどこまで行けたか

当初の1周 ＝ **開設 → 入金 → 配信 → 数字を見る → 1回改善 → 停止して振り返り**

| 工程 | 到達 | メモ |
|---|---|---|
| アカウント開設（オファー側・出稿側） | ✅ | Chaturbate＋HilltopAds 両方 |
| 入金 | ✅ | $100入金（最大の関門を突破） |
| LP作成・公開 | ✅ | 最小版 → コンプラ対応版に改修、Netlify公開 |
| 配信 | ❌ | **審査を突破できず到達せず** |
| 数字を見る | ❌ | 配信していないので数字は出ていない |
| 1回改善 | △ | 「配信の改善」ではなく「**審査落ちの改善**」を2回実施 |
| 停止して振り返り | ✅ | これ |

**到達点：「配信」の一歩手前で停止。** ただし、想定していなかった**「審査・ドメイン審査」という関門を実地で踏めた**のは、むしろ濃い学びだった。

## 3. 学べたこと（言語化）

**A. アービトラージの“地図”が腹落ちした**
広告ネットワーク（出稿＝お金を払う側）と、アフィリエイト（オファー＝お金をもらう側）は別物で、その2つを**クリックID／ポストバック**で繋ぐ——という全体構造を、実際に手を動かして理解できた。

**B. CPM と「予算 → 表示回数」の感覚**
CPM＝1000表示の値段。$0.74で約1000表示。$45で約6万表示。**安いGEOほど同じ予算で表示（チャンス）が増える**——この計算が体感できた。

**C. 損益分岐の考え方**
損益分岐CV率 ＝ CPC ÷ オファー単価。フィリピン（Tier-3）の無料登録は単価約$0.01なので、**そもそも金額的には赤字前提の学習設計**だと理解した。

**D. 一番の発見：本当のボトルネックは「審査」だった**
当初は「英語・海外決済での入金」が最大の関門だと考えていた。実際に越えてみると、**そこは突破できた**。代わりに立ちはだかったのが——

> **送り先URL（ドメイン）の審査。無料ホスティングのサブドメインが1社にブラックリスト判定され、「Unacceptable offer」で却下され続けた。**

つまり「稼ぐ手前で、“どのドメイン・どの送り先なら広告ネットワークが受け入れるか”という関門がある」と、身をもって知った。これは事前の競合調査でも「却下の約4割は送り先ページ起因」と出ていた通りだった。

**E. 規約・審査まわりの実務**
- 「中身の薄い転送だけのLP」「スマートリンク」は禁止 → 中身のあるページが要る
- 却下キャンペーンは「更新」では再審査に戻らない／クローンは2回目入金後のみ
- ドメインの健全性（VirusTotal 0/92）を出稿前に確認すべき

## 4. 数字の振り返り（テンプレ・正直版）

```
■ 数字
- 消化額（広告費）：$0  ← 配信に到達しなかったため
- 表示・クリック・CV：すべて 0（配信前で停止）
- 損益分岐CV率（理論値）：CPC ÷ $0.01 … Tier-3単価では現実的に黒字化困難と理解

■ 実際にかかったお金
- 入金手数料：約$3.2（約480円）
- 広告費：$0
- 残高 $96.8 は HilltopAds に温存（※引き出し可否は要確認。次回の出稿に使える）
- ドメイン代：未発生（独自ドメインは取得せず停止）
→ 実質コスト ≒ 480円で、広告運用の1周ぶんの実務を体験できた

■ 起きたこと（事実）
- 一番手こずった工程：広告審査（同じ「Unacceptable offer」で2回却下）
- 詰まった点：無料ホスティングのドメインが1/92でブラックリスト判定

■ 学び（解釈）
- 入金より「審査・ドメイン」が実際の関門だった
- 1要素ずつ潰す進め方（LP改修→再申請→VT確認）で原因を切り分けられた
- 「次にやるなら、最初から独自ドメイン＋SSLで出す」

■ 判定
- 1周完遂：配信は未到達。ただし学習目的は達成
- 黒字/赤字：実質コスト約480円（学習目的なのでOK）
- 続ける/やめる/別案へ：いったん停止して振り返り（再開の余地は残す）
```

## 5. 次にやるなら（再開時の一手）

1. **独自ドメインを取得**（年100〜1,500円・`.xyz`等）し、SSL付きでLPを乗せ替える。VirusTotalで 0/92 を確認してから出す。
2. それで新規キャンペーンを作り、再申請。通れば**ようやく「配信 → 数字を見る」**に進める。
3. 配信が回ったら、当初設計どおり **Day 5「死んだ枠をブラックリスト化」** を1回やって振り返る。
4. 残高 $96.8 はHilltopAdsに残っているので、**再開時の広告費に使える**（引き出し可否はサポートに要確認）。

## 6. 総評

「最小コストで1周を学びきる」という目標に対して：

- **配信・数字までは届かなかった**が、その手前の**実務（開設・入金・LP・公開・審査・却下・原因特定）を一通り実体験**できた。
- とくに **“本当の関門は審査とドメインだった”** という発見は、机上の調査では得られない、今回いちばんの収穫。
- 実質コスト **約480円**。授業料としては破格に安く、よく回収できた1周だった。

> 当初の方針「利益最大化が目的ではなく、最小コストで1周を学びきる」——その精神には、十分に沿えている。
