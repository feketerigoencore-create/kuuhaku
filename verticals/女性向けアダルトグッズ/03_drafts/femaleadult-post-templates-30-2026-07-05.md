# 投稿テンプレ30本(非露骨)｜2026-07-05

X と Threads 用の投稿雛形です。Threads はさらにウェルネス寄りで穏当にします。
◯は具体語に置き換えます。効能は断定せず、体験は一般化します。
リンクは受け皿へ。Threadsは本文リンクを避け「プロフから」を定型にします。

## 使い方

- 型: A共感 / B Tips / C比較 / D誤解 / E一次の声 / F場面 / G数字 / H失敗回避 / I問いかけ。
- PR表記は、リンクを含む投稿・告知に必ず添えます。
- 露骨語・効能断定・煽りは使いません。「個人差」を添えます。
