# X/Threads プロフィール・固定投稿 文面案｜2026-07-05

X と Threads のプロフィールと固定投稿の文面案です。
非露骨・ウェルネス文脈を維持し、PR方針と受け皿導線を明示します。
公開前に各媒体の文字数・リンク規約を再確認してください(要確認)。

## 0. 共通の考え方

- 名乗りは「セルフケア/フェムケアの選び方を穏当にまとめる人」。
- 露骨語を使わず、悩み解決・情報整理のトーンにする。
- プロフに受け皿(LP)リンク、固定で導線と方針を明示する。
- PR方針(広告を含む旨)を分かる場所に置く。

## 1. Xプロフィール案

### 案A(情報整理タイプ)
- 名前: 「フェムケアの選び方メモ」
- bio: 「初めてのセルフケア用品を、静音・お手入れ・素材で穏やかに比較。医療的効果は断定しません。個人差あり。PRを含みます。18歳以上向け。」
- リンク: 受け皿LP。
- 補足: センシティブ設定を正しく。露骨化しない。

### 案B(共感タイプ)
- 名前: 「そっと、セルフケアの話」
- bio: 「聞きにくいお悩みと、選び方の要点をやさしく整理。効果は約束しません(個人差)。PR含む/18歳以上。詳しくは↓」
- リンク: 受け皿LP。

## 2. Threadsプロフィール案

### 案A(ウェルネスタイプ)
- 名前: 「フェムケアとセルフケアのはなし」
- bio: 「自分をいたわる時間の作り方と、選ぶ視点をやさしく。一般情報のみ・効果は断定しません。18歳以上向け。詳細はリンクから(PR)。」
- リンク: プロフに受け皿LP(本文リンクは使わない)。

### 案B(会話タイプ)
- 名前: 「セルフケア、どうしてる?」
- bio: 「小さな悩みと工夫をゆるく交換する場所。医療的効果は言いません(個人差)。18歳以上/PR含む。まとめは↓」
- リンク: プロフに受け皿LP。

## 3. X固定投稿案

```
はじめまして。セルフケア用品の「選び方」を穏やかに整理しています。
扱うのは静音・お手入れ・素材・価格などの客観的な視点です。
医療的な効果は断定しません(個人差があります)。
広告(PR)を含みます。18歳以上の方向けです。
選び方の要点はこちらにまとめました → (LPリンク)
```

## 4. Threads固定投稿案

```
はじめまして。自分をいたわる時間と、フェムケアの選び方をやさしくまとめています。
ここでは一般的な情報だけをお話しします。効果を約束するものではありません(個人差)。
18歳以上の方向けの内容です。広告(PR)を含みます。
詳しいまとめは、プロフィールのリンクからご覧いただけます。
```

## 5. 運用メモ

- Xは固定に受け皿リンクを含めてよい。Threadsは本文リンクを避け、プロフへ誘導。
- 名前・bioは文字数超過に注意(公開前に実機で確認)。
- 露骨表現・効能断定・煽りを入れない。PRと年齢配慮を必ず残す。
