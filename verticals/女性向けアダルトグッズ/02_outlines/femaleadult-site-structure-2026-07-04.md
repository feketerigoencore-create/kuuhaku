# 女性向けアダルトグッズ サイト設計｜2026-07-04

このサイトの土台を定義します。カテゴリ・URL・固定ページを決め、記事制作の基準にします。
土台: [[00_MASTER戦略_主軸_2026-07-04]]／型: [[asmr-site-blueprint-and-review-template-2026-07-04]]

## サイトの前提

媒体は WordPress と独自ドメインです。理由は、資産化と SEO に強く、規約リスクを避けられるためです。
対象は初心者の女性です。不安と恥ずかしさを解消し、安全に選べる状態へ導きます。
全ページに PR 表記を置きます。医療効能は断定しません。露骨な表現は避けます。

## カテゴリ設計

カテゴリは5つに絞ります。初心者の導線(知る→選ぶ→比べる→決める→迷いを解く)に沿わせます。

- 基礎知識(`/basic/`) … セルフプレジャーとは、体の仕組み、衛生の基本。集客の入口です。
- 選び方(`/choose/`) … ローターの選び方、安全な素材、静音・防水。比較検討の核です。
- ブランド比較(`/brand/`) … iroha など指名検索を回収します。
- ランキング(`/ranking/`) … 目的別・予算別のおすすめ。セール期に訴求します。
- お悩みQ&A(`/qa/`) … 音・バレ・収納・衛生など1問1答で不安を解きます。

### URL 設計の方針

URL は英小文字とハイフンで統一します。日本語スラッグは使いません。
形式は `/<カテゴリ>/<記事スラッグ>/` です。階層は2段までにします。

- 基礎知識: `/basic/selfpleasure-toha/`、`/basic/eisei-kihon/`
- 選び方: `/choose/rotor-hajimete/`、`/choose/body-safe-material/`
- ブランド比較: `/brand/iroha-hikaku/`
- ランキング: `/ranking/yosanbetsu/`、`/ranking/mokutekibetsu/`
- お悩みQ&A: `/qa/oto-bare/`、`/qa/eisei-teire/`

### カテゴリ運用のルール

1記事は必ず1カテゴリに属させます。重複所属は避けます。
タグは検索意図(初心者/静音/防水/予算別など)で補助的に付けます。
パンくずは カテゴリ→記事 の2段で表示します。

## 固定ページ一覧

固定ページは信頼設計の柱です。運営者が誰でも成立する仕掛けを、ここで宣言します。

- 運営者・編集方針(`/about/`) … 誰が何の方針で書くか。一次の声と出典を重視する姿勢を示します。
- 監修について(`/supervision/`) … 監修の有無と範囲。可能なら監修者を明示します。【監修者プレースホルダ】
- PR・広告方針(`/ad-policy/`) … アフィリエイトを含む旨と、順位づけの基準を説明します。
- プライバシーポリシー(`/privacy/`) … 個人情報とアクセス解析の扱いを記載します。
- 年齢確認・ゾーニング(`/age-check/`) … 18歳以上向けの案内と閲覧範囲を示します。
- お問い合わせ(`/contact/`) … 連絡窓口。指摘や訂正依頼を受ける導線です。

### 各固定ページの要旨

運営者・編集方針では、初心者の不安解消を目的に置くことを述べます。体験の断定を避け、公開レビューと仕様で語る方針を明記します。
監修についてでは、健康や体の情報を扱う範囲で監修や参照元を示します。医療行為の助言ではないと断ります。
PR・広告方針では、報酬が順位に影響しない基準を説明します。景表法とステマ規制への対応を明記します。
プライバシーポリシーでは、解析ツールと Cookie の利用、問い合わせ情報の扱いを説明します。
年齢確認・ゾーニングでは、対象年齢と、露骨表現を避ける編集方針を案内します。
お問い合わせでは、事実誤りの指摘を歓迎する姿勢を示します。訂正の記録方針も添えます。

## トップページの構成

トップは初心者を迷わせない導線にします。上から順に、次の要素を置きます。

1. サイトの一行説明(恥ずかしさを手放し、安全に選べる)。
2. はじめての方へ(入口記事3本へのリンク)。
3. カテゴリ入口(5カテゴリのカード)。
4. 人気・おすすめ(ランキングへの導線)。
5. 編集方針と PR 方針への内部リンク。

## 内部リンクの基本設計

流れは 入口→選び方→比較→ランキング です。各記事末に「次に読む」を置きます。
基礎知識から選び方へ、選び方から比較へ、比較からランキングへ送ります。
詳細な集客導線は [[femaleadult-traffic-plan-2026-07-04]] に定義します。

## 信頼設計(調査班・確定)

運営者が男性でも成立させるため、競合の信頼設計を踏襲します。[出典: 信頼ベンチマーク/競合分析]

- 監修と商品推奨を分離します。iroha 方式で「監修者は商品を推奨しない」旨を明記します。
- 一次の声を構造化して載せます。公開レビュー・アンケートを出典と年次つきで引用します。
- 男性運営であることを編集方針で開示し、透明性そのものを信頼材料にします。
- 差別化は「中立のブランド横断比較」。競合(iroha=自社のみ/EC=自社取扱)が薄い領域です。

## 収益設計の前提(調査班・確定)

- 主力ASP: NLS(最大25%・R18前提)＋ラブトリップ(女性向け専門・報酬率は要確認)。
- 補助: Amazon/楽天で一般流通品(iroha 等)を回収。ただしアダルトサイト登録は不可で、露骨表現を排す設計が必須。
- A8.net はアダルト禁止。afb は露骨表現サイト不可。主力2〜3社に絞ります。[出典: アフィリ条件メモ]

## 制作の共通ルール(再掲)

- 敬体で書きます。一文は60字以内にします。見出しは h2→h3 のみにします。
- 全記事に PR 表記の枠と「一次の声」枠を入れます。
- 商品名・価格・一次の声は【プレースホルダ】にします。確定は人間と調査班に委ねます。
- 医療効能は断定しません。健康の話は一般情報に留めます。
