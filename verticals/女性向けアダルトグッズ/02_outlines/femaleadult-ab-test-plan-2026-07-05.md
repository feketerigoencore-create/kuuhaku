# 初月 投稿型A/B設計｜2026-07-05

初月に投稿型を比べ、伸びる型へ配分を寄せるための設計です。
比べる軸は「お悩み共感」「Tips」「比較の要点」の3型です。
露骨化はしません。全型ともウェルネス文脈を維持します。

## 0. 目的

- どの投稿型が「保存・返信・送客」に効くかを、初月で見極める。
- 媒体差(X/Threads)も同時に見る。勝ち型×勝ち媒体に寄せる。

## 1. 比較する3型

- 型A お悩み共感: 「◯◯なとき、同じ悩みの人は多いです。」
- 型B Tips: 「選ぶときに見るのは静音・お手入れ・素材の3つ。」
- 型C 比較の要点: 「AとBの違いは◯◯。詳細はプロフから。」

## 2. 設計(公平に比べる条件)

- 各型を各媒体で週3投稿、計4週。投稿時間帯は概ね揃える。
- 1投稿=1型に限定し、混ぜない(効果の切り分けのため)。
- 受け皿リンクの出し方は媒体ルールに従う(Threadsは週3・本文外)。
- 画像・長さ等の体裁はできる範囲で揃える。

## 3. 測る指標

| 指標 | 意味 | 主に見る媒体 |
|---|---|---|
| 保存数 | 実用性・後で見たい度 | X |
| 返信数 | 会話の起きやすさ | Threads |
| プロフ表示 | 送客の手前 | 両方 |
| 受け皿クリック | 送客の実数 | 両方 |

## 4. 記録テンプレ(週次)

| 週 | 媒体 | 型 | 投稿数 | 保存 | 返信 | プロフ表示 | クリック |
|---|---|---|---|---|---|---|---|
| 1 | X | A |  |  |  |  |  |
| 1 | X | B |  |  |  |  |  |
| 1 | X | C |  |  |  |  |  |
| 1 | Threads | A |  |  |  |  |  |
| 1 | Threads | B |  |  |  |  |  |
| 1 | Threads | C |  |  |  |  |  |

以降 週2〜4 も同様に記入します。

## 5. 判定と反映

- 4週後、媒体×型で「クリック/投稿」が高い組を勝ちとする。
- 翌月は勝ち組の比率を上げる(例: Threads×共感が強ければ増やす)。
- 弱い組は型を微修正して再検証、またはやめる。
- 結果は [[femaleadult-success-criteria-2026-07-05]] の判定に反映。

## 6. 注意

- サンプルが少ないと偶然に振れます。断定せず傾向で見ます。
- 規約(週3リンク等)を崩してまで比較条件を揃えないこと。
- 露骨化での「一時的な伸び」は採用しない(凍結・審査リスク)。
