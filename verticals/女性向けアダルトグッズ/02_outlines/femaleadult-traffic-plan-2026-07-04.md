# 集客プラン｜内部リンク設計とSNS初月カレンダー

無料流入を軸にした集客設計です。SEO を主軸に、Pinterest と X を補助にします。
土台: [[00_MASTER戦略_主軸_2026-07-04]]／サイト設計: [[femaleadult-site-structure-2026-07-04]]

## 前提

- アダルト扱いのため、広告出稿と一部SNSに制限があります。要確認。
- そのため無料流入(SEO・Pinterest・X)を中心に据えます。
- トーンはウェルネス文脈で統一します。露骨表現は避けます。

## SEO 内部リンク設計

読者の段階に沿って、入口→選び方→比較→ランキングへ送ります。

### 導線の基本フロー

1. 基礎知識(入口) … 「セルフプレジャーとは」で集客します。
2. 選び方 … 「ローターの選び方」へ送り、比較軸を理解させます。
3. ブランド比較 … 「iroha 比較」で指名検索を回収します。
4. ランキング … 「予算別・目的別」で決定を後押しします。
5. お悩みQ&A … 各段階から不安を解消し、離脱を防ぎます。

### リンクの張り方ルール

- 各記事末に「次に読む」を2〜3本置きます。
- 選び方記事から比較・ランキングへ必ずリンクします。
- Q&A から選び方・比較へ戻す動線を作ります。
- アンカーテキストは内容が分かる語にします。「こちら」は避けます。

### ハブとなる記事

- 選び方(ローター)を中心ハブにします。入口とQ&Aから集め、比較へ送ります。
- ランキングを収益ハブにします。比較記事から集約します。

### 優先キーワード(調査班・◎=競合が手薄)

キーワードマップで、公式・女性誌と正面衝突しにくい語を優先します。[出典: キーワードマップ]

1. body safe 素材 / 安全 素材 見分け方(空きが大、信頼設計の核)。
2. ローター 初めて 選び方(静音/防水/価格/素材のCVハブ)。
3. 静音 ローター / ローター 音 バレない(不安起点で流入)。
4. iroha 比較 どれがいい(指名検索の受け皿)。
5. セルフプレジャー 恥ずかしい / 初心者 やり方(上流集客)。
6. ローター 収納 バレない / 洗い方 手入れ(空きが大、回遊)。

差別化の軸: 競合(iroha=自社のみ/EC各社=自社取扱)が薄い「中立のブランド横断比較」を主戦場にします。yoi(集英社)の閉鎖で医療監修の信頼枠に空きがあります(要確認)。[出典: 競合分析]

## Pinterest 運用(補助・主力の無料流入)

- 女性の保存文化と相性がよく、ウェルネス文脈で運用できます。
- 縦長画像で「選び方」「Q&A」を保存されやすくします。
- 露骨な画像は使いません。図解・チェックリスト中心にします。
- 各ピンは対応記事へリンクします。

## X 運用(補助)

- ウェルネス文脈で穏当に発信します。煽りません。
- 初速と返信を重視します。詳しい型はポケカの運用ナレッジを参照します。
- 誘導は控えめにし、プロフィールやリプで記事へ送ります。

## KPI(初月)

- 表示回数と検索順位(初心者ロングテール)。
- 記事→比較→ランキングの回遊率。
- Pinterest の保存数、X の返信・ブックマーク。

## 広告出稿の制限(調査班・確定)

- Google 広告: 検索では配信できる場合があるが、ディスプレイ・YouTube・AdMob は禁止。地域・年齢・セーフサーチに依存。
- Meta(FB/Instagram): 成人向け商品の宣伝は原則禁止(家族計画・避妊の例外のみ)。
- X: 成人向けの性的コンテンツの広告掲載は禁止。
- 結論: 有料出稿は不可・不安定。無料流入(SEO・Pinterest保存・Xウェルネス発信)を主軸にします。[出典: コンプラメモ]

## 残る要確認

- Pinterest の成人向け規約(一次確認が未完)。出稿・運用前に再確認。
- 各SNSのアフィリンク可否とゾーニング要件。
- 実検索ボリューム・難易度(キーワードプランナー等で裏取り)。
