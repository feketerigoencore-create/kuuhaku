# X/Threads(Meta) 成人向け・商用・リンク規約の線引き｜2026-07-05

X と Threads で女性向けグッズをどこまで発信できるかを、条項ベースで整理します。
規約は改定が多いです。投稿・出稿の直前に必ず公式を再確認してください。
取得時点は 2026-07-05 です。未確認は「要確認」と明記します。

## 0. 結論(先に線引き)

- Threads は実質 SFW(健全)前提です。露骨表現・グッズの直接推しは避けます。
- Threads はフェムケア/ウェルネスの一般情報に限定します。リンクは本文以外で週3まで。
- X は投稿は踏み込めますが、収益・拡散の面で制約が増えました(下記 ACC)。
- どちらも成約はSNS外の受け皿(LP/比較記事)で取る設計が安全です。

## 1. X(旧Twitter)の最新ルール

X は成人向けの投稿自体は許容しますが、2026年に分類と収益の扱いが変わりました。

### 分類とセンシティブ設定

- 従来の2択(Sensitive/Not)から3段階へ変更との情報があります(要確認)。
- 3段階は Sensitive / Adult / Explicit の区分とされます。
- 成人向けを定常投稿するなら「投稿にセンシティブなメディアを含む」設定を有効にします。
- 設定 > プライバシーと安全 > あなたの投稿、から切り替えます。
- 未設定で成人向けを投稿すると自動の是正対象になるとされます。

### ACC(Adult Content Creator)プログラム

- Adult/Explicit 区分の投稿を複数回する場合、ACC 登録が要るとの情報です(要確認)。
- 登録には本人確認(政府発行ID)・年齢確認(18歳以上)が必要とされます。
- ACC 登録アカウントは「おすすめ」「トレンド」「プロモ枠」から自動除外されるとされます。
- 発見経路は検索・フォロワーのTL・直リンクに限られるとされます。

### 収益化(重要)

- X の収益化プログラムは成人向けを対象外とするとされます。
- 「成人向けで集客」か「非成人で収益化」かの二者択一になりやすいです。
- 本ジャンルは X 内の収益に頼らず、受け皿経由の外部収益(ASP)を前提にします。

### 女性向けグッズの線引き(X)

- ウェルネス/セルフケア文脈の非露骨な発信は投稿可能と考えられます。
- ただし露骨さを上げると Adult/Explicit 判定→ACC・除外の負担が増えます。
- 客層は男性寄りとされ、女性向けはリーチが弱い可能性があります(要検証)。
- 方針: X も原則ウェルネス寄りの非露骨で運用し、露骨化はしない。

## 2. Threads(Meta)の最新ルール

Threads は Meta のコミュニティ規約に従い、実質 SFW の会話プラットフォームです。

### コンテンツの扱い

- 成人向け・露骨な性的表現は Threads の想定外で、配信は別媒体が適切とされます。
- 二次情報でも「アダルトはThreads不向き、健全会話に使え」との整理が一般的です。
- グッズの露骨な推しは凍結リスクがあるため避けます。

### 広告(参考・出稿する場合)

- 性・生殖の健康/ウェルネス広告は避妊・家族計画等に限られるとされます。
- その場合も18歳以上ターゲットで、性的快楽に焦点を当てないことが条件とされます。
- 快楽目的の性的グッズ・成人向けサービスの広告は禁止とされます。
- 本ジャンルは Meta広告に不向きです。無料投稿(オーガニック)で運用します。

### リンクの扱い

- Threads 本文へのアフィリンク常時掲載は避けます(凍結・スパム判定リスク)。
- リンクはプロフィール または 自己リプライに置き、露出は週3までを目安にします。
- 「詳細はプロフィールから」を定型化し、本文は会話・共感に徹します。
- 要確認: Threads の商用リンク・アフィリンクに関する明文規約は公式で未特定です。

### 女性向けグッズの線引き(Threads)

- 許容されるのは「フェムケア/セルフケアの一般的ウェルネス情報」までです。
- 露骨な語・画像、快楽の直接訴求、購入の直接煽りは避けます。
- お悩み共感・生活の工夫・選ぶ視点(静音/お手入れ等)の穏当な話題に限ります。

## 3. 媒体比較(役割分担の根拠)

| 観点 | X | Threads |
|---|---|---|
| 成人向け投稿 | 可(3段階分類・ACC条件) | 不可寄り(SFW前提) |
| 露骨さの上限 | 中(上げると負担増) | 低(ウェルネスのみ) |
| リンク | 比較的自由(受け皿へ) | 本文NG/プロフ・自己リプ/週3 |
| 拡散・発見 | ACCで除外あり・検索/フォロワー中心 | 会話・返信で広がる |
| 収益化(媒体内) | 成人は対象外 | 想定なし |
| 客層 | 男性寄り(要検証) | 女性・生活層が比較的多い(要検証) |
| 役割 | やや踏み込む集客・比較の要点 | やわらかい入口・共感で裾野拡大 |

## 4. 運用チェック(投稿前)

- [ ] Threads は非露骨・ウェルネス限定。露骨語・快楽の直接訴求を書いていない。
- [ ] Threads のリンクは本文外(プロフ/自己リプ)、週3以内。
- [ ] X はセンシティブ設定を正しく設定済み。露骨化していない。
- [ ] 成約は受け皿経由。SNS単体アフィリに依存していない。
- [ ] PR表記・医療非断定・年齢配慮を満たしている。
- [ ] 凍結に備え、受け皿(自分の資産)へ必ず送る導線がある。

## 5. 要確認(次アクション)

- X の3段階分類・ACC 要件の一次(公式ヘルプ)での確認。
- Threads の商用/アフィリンクに関する明文規約の有無。
- Meta のウェルネス広告ポリシー最新版(改定頻度が高い)。
- 実測での客層(X が男性寄りか、Threads が女性寄りか)の検証。
