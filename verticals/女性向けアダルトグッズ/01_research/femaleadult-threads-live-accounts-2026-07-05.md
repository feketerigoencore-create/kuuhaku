# Threads フェムケア/ウェルネス発信者の実測｜2026-07-05

Threads で非露骨に伸ばす発信者の型と、リンク導線を採取します。
Threads は SFW 前提のため、収集対象はウェルネス文脈に限ります。
個別数値は変動が速く、断定しません(要確認)。

## 0. 前提(Threadsの制約)

- Threads は実質 SFW です。露骨表現・グッズ露骨推しは対象外にします。
- リンクは本文NGで、プロフ/自己リプに置くのが一般的です(週3目安)。
- よって観測は「非露骨で会話を伸ばす型」と「リンクの逃がし方」に絞ります。

- 受け皿: ブログ/LP/note/Instagram のどれへ送るか。

## 2. 観測の起点(型の参考)

- フェムケア啓発・メディア系: 生理/PMS/デリケートゾーンの一般情報で会話を作る型。
  露骨度: 低。リンク: プロフ経由。→ 「共感＋一般情報」で裾野を広げる参考。
- ウェルネス個人(セルフケア全般): 睡眠・リラックス・自分をいたわる話題に絡める型。
  → グッズを直接出さず、生活文脈から受け皿へ流す参考。
- ブランド公式(フェムケア一般流通): 世界観・使い方の穏当な発信。
  → 「売り込まずに信頼を作る」トーンの参考。

注: Threads は国内でも会話・共感が伸びやすい傾向とされます(要検証)。
具体アカウントは手動で10件採取し、上表に追記します。

## 3. 採取の手順(手動観測)

1. Threads で「フェムケア」「セルフケア」「デリケートゾーン」等を検索。
2. 非露骨で返信が付く投稿を開き、上表に記入する。
3. プロフ・固定のリンク先(受け皿)と、リンクの置き方を確認する。
4. 伸びている投稿の共通点(問いかけ/共感/保存性)をメモする。
5. 露骨な発信は記録対象外にする。

## 4. 見るべき観点(仮説)

- Threads は「問いかけ→返信の往復」で伸びやすい仮説(会話量が指標)。
- 女性・生活層が X より多い可能性があり、女性向けと客層が合う仮説(要検証)。
- 直接販売色を消すほど、共感と保存が伸びやすい仮説。
- リンクを週3に抑えても、プロフ導線が明確なら送客は成立する仮説。
