# ポケカ相場ブログ X×ブログ併用 運用ナレッジ（2026-07）

X とブログを併用して回すために必要な知識と最新機能を、一次確認できた範囲でまとめます。仕様と数値は変動が速いので、断定前に各ソース（末尾）で再確認してください。関連: [[pokeca-運用設計_週次オペレーション]]。

> 前提の再確認: 稼ぐのはカード売買ではなく情報発信です。X は集客、ブログは資産。X 収益（分配金）はおまけで、当てにしません。

## X の要点（2026年アルゴリズム）

いま伸ばすうえで押さえる点です。

- **返信（会話）が最重視**: 返信はいいねより大幅に重く評価されます（各所で「いいねの数十倍」との集計）。会話が生まれる投稿が伸びます。
- **初速が命**: 投稿後30分ほどの反応で拡散するか決まります。反応が付きやすい時間帯に出します。
- **ネイティブ形式が有利**: 画像・動画・投票など、Xの中で完結する形式が優遇されます。テキスト先行の投稿も強いです。
- **Premium で到達が伸びる**: Premium 加入者は非加入より到達が優遇されるとされます（2〜4倍との集計）。
- **ニッチ集中が効く**: 同じ興味クラスターから反応を集め続けると、その集団へ到達が広がります。ポケカ相場に一貫特化するほど権威が複利で効きます。
- **AI ランキング化**: 2026年にGrokベースの推薦へ刷新されたと報じられています。投稿内容そのものが読まれて配信先が決まる前提で、中身の質が効きます。

（出典: Sprout Social, SocialBee, OpenTweet ほか。末尾ソース参照）

## X の収益化条件（2026年・参考）

Creator Revenue Sharing（返信・表示ベースの分配）の主な条件です。**当面は目標にせず、集客の副産物**と捉えます。

- X Premium（$8/月）以上に加入。
- **verified フォロワー 500人以上**（Premium 認証済みのフォロワー）。
- **直近3ヶ月で500万インプレッション以上**（Premium/認証ユーザーの表示が対象。無料アカウントや検索経由は原則カウント外）。
- Stripe 連携・アカウントが規約順守で良好な状態。
- 支払いは隔週、最低 $30。届かない分は次回に繰り越し。

> 円建て価格や国別の細目は変わり得ます。**要確認**。この条件到達は早くて数ヶ月〜。最初はフォロワー数より「型の定着」を優先します。

## 外部リンクの扱い（併用で最重要）

ブログへ送客するうえで、Xの外部リンク評価は避けて通れません。ここは見解が割れています。

- 長らく「外部リンク付き投稿は到達が下がる（50〜90%減との指摘）」とされてきました。
- 一方で「2025年10月にリンクペナルティは撤廃された」との報道もあります。**現状は流動的・要確認**。
- 割れていても一貫するのは、**ネイティブ投稿がリンク投稿より伸びやすい**という点です。

ポケカ運用への落とし込み（ペナルティ有無に関わらず有効）:

1. **本文はリンクなしで価値を完結**させます。相場の要点・図・数字だけで読ませます。
2. リンクは**返信（1つ下）か2ツイート目**に置きます。本文で関心を引いてから誘導します。
3. **誘導はブログ更新日だけ**。毎回リンクを貼らないほうが、平時の到達を守れます。
4. 単発リンク連投より、**定点観測データそのものをXに載せて話題化**し、詳細をブログに、が基本線です。

## 使う機能・使わない機能（2026）

- **ロング投稿 > スレッド**: 単発のロング投稿が、複数ツイートのスレッドより配信面で優遇される傾向。長め解説はロング投稿でまとめます。
- **予約投稿**: 週次バッチ（日曜に翌週分）を予約で流します。X 純正の予約か外部ツール（Buffer, SocialPilot 等）を利用。純正で足りるかは**要確認**。
- **アナリティクス**: 表示・エンゲージ率・プロフィール訪問を確認。**いいねより「返信・ブックマーク」を主指標**にします。キーワード分析ツール「Radar」も一部に開放。
- **Communities は終了**: X Communities は2026年5月30日に恒久削除と告知。**コミュニティ機能に依存しない**運用にします。

## ブログ側の最新SEO（2026）

Google の方向性は、ポケカの独自性戦略と噛み合います。

- **情報の独自性が決定打**: 「どこにも存在しない情報」を含むかが再重み付けされました。→ あなたの**週次定点観測データ**がまさにこれに当たります。
- **E-E-A-T（経験・専門性・権威・信頼）**: AI飽和のSERPでより重要に。**実名の運営者と実績（ポケカ歴・売買記録）**が効きます。物語ブランドがそのまま権威の裏付けになります。
- **ドメイン単位の網羅性**: 1テーマを網羅的に深掘りするサイトが、浅く広いサイトに勝ちます。→ ポケカ相場に一点集中が正解。
- **AI下書き＋人間の差別化**: AIは調査・構成・初稿に使い、**視点・独自データ・実例・判断は人間が上書き**。これが順位に効きます（CLAUDE.md の「Claude下書き→自分の相場観で上書き」と一致）。

（出典: Google Search Central, evertune, Rankability ほか）

## AdSense 審査の要件（2026・目安）

クリック広告を入れるなら、審査は記事が貯まってからです。

- **独自・人間主体の記事 15〜25本**（各800〜1500字以上が目安。情報源で幅あり）。
- 必須ページ: 運営者情報・お問い合わせ・**プライバシーポリシー（AdSenseとCookieに言及）**・免責。
- **HTTPS 必須**・モバイル対応・ナビが明快。
- E-E-A-T（実名運営・経歴・独自の知見）が見られます。
- 国によってはドメイン年齢の目安あり。最低トラフィックは不要とされます。**要確認**。

必須ページの文面雛形は [[pokeca-wordpress-setup-guide]] の STEP5 に用意済みです。プライバシーポリシーに AdSense 明記を足すだけで要件に寄せられます。

## 併用の全体導線（X → ブログ → 収益）

回路はシンプルにします。

1. **X で集客**: 定点観測・物語・解説で会話を作り、ニッチ権威を貯めます（本文はリンクなし）。
2. **更新日だけブログへ誘導**: 返信/2ツイート目にリンク。関心を引いてから送客。
3. **ブログで信頼と流入を資産化**: 独自データ＋E-E-A-Tで検索流入を育てます。
4. **収益化**: まずアフィリ（トレカ系ASP）→ 記事15本超でAdSense → 後期に自分の有料note/Kindle。X分配金はおまけ。
