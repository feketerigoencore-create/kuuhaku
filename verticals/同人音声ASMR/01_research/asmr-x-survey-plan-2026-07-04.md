# X生調査プラン(拡張接続後に自動実行)｜2026-07-04

Claude in Chrome 接続後、このプランに沿ってX内を巡回し、下の採取テンプレを埋めます。
目的: 同種アカウントの実測(投稿型・頻度・エンゲージ・導線)で競合分析を確定する。

## 0. 前提・注意
- R18閲覧・年齢認証は本人環境で行う。露骨表現の収集はしない(分析目的の構造把握のみ)。
- 収集は公開情報のみ。数値は目視の概算で「時点」を添える。

## 1. 検索クエリ集(Xの検索窓に入れる)
基本:
- 同人音声 おすすめ / ASMR おすすめ 音声作品 / DLsite 音声 感想
- 安眠 ASMR / 耳かき ボイス / バイノーラル 耳舐め / 催眠音声 感想
- 同人音声 レビュー / 音声作品 紹介 / ベスト同人音声
絞り込み(高エンゲージ・新しめ):
- 例) 同人音声 おすすめ min_faves:50
- 例) ASMR 音声作品 min_retweets:20 since:2026-01-01
- 例) DLsite 音声 レビュー filter:links
属性別:
- メスガキ 音声 / 女性向け シチュボ / オナサポ おすすめ(トレンド・空白地帯の確認)
