# 表現ガイド 薬機法・景表法・ステマ規制(女性向けラブグッズ/雑貨アフィリ)

作成日: 2026-07-05 / 対象: 女性向けラブグッズ(雑貨)のアフィリ記事

## このガイドの位置づけ

本稿は合法なアフィリ運営のための一般情報です。個別の表現が違法かどうかは、文脈や商品分類で結論が変わります。最終判断は弁護士や薬事・広告の専門家に確認してください。法令や運用基準は改正されます。掲載URLと最新版を都度確認してください(要確認)。

対象商品はラブグッズのうち「雑貨」に分類されるものを想定します。医薬品・医薬部外品・化粧品は本稿の対象外です。ローションやジェルなど、成分や用途によっては化粧品や医薬部外品に該当する場合があります。分類が不明な商品は、表現を検討する前に分類自体を確認してください(要確認)。

## 3つの規制の全体像

女性向けラブグッズのアフィリ記事に関わる主な法規制は3つです。薬機法、景品表示法(景表法)、そして景表法に含まれるステマ規制です。それぞれ守るべき点が異なります。

- 薬機法: 体への効能効果を書けるかの問題です。
- 景表法(優良誤認・有利誤認): 品質や価格を大げさに書いていないかの問題です。
- ステマ規制: 広告であることを隠していないかの問題です。

## 薬機法: 効能効果の断定はなぜNGか

### 雑貨は体への効能効果を一切うたえない

薬機法は、承認を受けていない製品について医薬品的な効能効果を標榜することを禁じています。雑貨は効能効果の承認を受けていません。よって、体への効果を示す表現は書けません。

化粧品や医薬部外品なら一部の効能を書けますが、雑貨はその範囲にも入りません。医薬部外品で認められる「疲労回復」「肩のこりをほぐす」なども、雑貨では書けないとされています(要確認)。

### 「何人も」が規制対象

薬機法の広告規制(誇大広告等の禁止)は「何人も」を対象とします。メーカーや薬剤師だけでなく、広告代理店、ASP、メディア運営者、そしてアフィリエイターも対象です。「自分は紹介しているだけ」という理由では免れません(要確認: 個別事案は専門家に確認)。

なお、アフィリエイターの記事であっても、その広告の責任は原則として広告主が負うと解されています。だからといってアフィリエイター側が無関係になるわけではありません。書き手として自衛が必要です。

### 断定がNGな理由

「治る」「効く」「改善」「予防」などは、体の状態を変える医薬品的な効果を示します。雑貨でこれを書くと、承認外の効能効果の標榜にあたるおそれがあります。

「安心」「安全」も注意が必要です。安全性を保証する表現は、医薬品等でも原則禁止とされています。雑貨でも体への安全性を保証する印象を与える書き方は避けます。

### NG→OK言い換え(薬機法)

体の効果ではなく、気分・雰囲気・使用シーンで書くのが基本方針です。

| NG表現 | NGの理由 | OKの言い換え例 |
| --- | --- | --- |
| 冷え性が治る | 疾病の治療効果の標榜 | 温かい使い心地を楽しめます |
| 生理痛に効く | 疾病への効果の標榜 | リラックスしたい時間に |
| 不感症を改善する | 身体機能への効果の標榜 | 自分のペースで向き合う時間に |
| 病気を予防する | 疾病予防効果の標榜 | 日々のセルフケアの習慣に |
| 疲労回復に | 医薬部外品的効能で雑貨は不可 | 一日の終わりのくつろぎに |
| ホルモンバランスを整える | 身体機能への一般的効果 | 自分をいたわる時間の演出に |
| 安全だから安心して使える | 安全性の保証 | 使い方の説明をよく読んでご使用ください |
| 医師も推奨 | 医療関係者による効果の保証 | (削除。体験談も注釈付きで不可) |

体験談で効果をうたう書き方も、薬機法上NGとされます。「使ったら痛みが消えた」のような感想も、効果の標榜と見なされるおそれがあります(要確認)。

## 景表法: 優良誤認・有利誤認・打消し表示

### 優良誤認

実際より品質が著しく優れていると誤認させる表示です。根拠のない最上級表現が典型例です。

「最強」「絶対」「No.1」などは、合理的根拠(調査データ等)がなければ優良誤認にあたるおそれがあります。消費者庁は「顧客満足度No.1」などのNo.1表示について実態調査を行い、根拠を欠く事例への執行を強めているとされます(要確認: 最新の調査状況は消費者庁で確認)。

### 有利誤認

実際より取引条件が著しく有利だと誤認させる表示です。価格・割引・数量の見せ方が問題になります。「常に最安値」「他店より必ず安い」などは根拠がなければ有利誤認にあたるおそれがあります。

### 打消し表示の注意

「効果には個人差があります」のような注記を打消し表示と呼びます。強調表示(大きく目立つ主張)と打消し表示のバランスが問題になります。

- 本文で強い印象を与え、注記を小さく薄く目立たない位置に置くと、打消しとして機能しないと判断されるおそれがあります。
- 打消し表示は、消費者が実際に読める大きさ・色・配置にします。
- そもそも根拠のない主張を注記でごまかすことはできません。

### NG→OK言い換え(景表法)

| NG表現 | NGの理由 | OKの言い換え例 |
| --- | --- | --- |
| 効果は最強 | 根拠なき最上級+効果標榜 | 使い心地の好みは人により異なります |
| 絶対に満足できる | 根拠なき断定 | 満足したという声もあります(出典明記) |
| 楽天No.1(根拠記載なし) | 根拠を示さないNo.1 | (調査元・期間・範囲を併記できる時のみ使用) |
| 100%リピート | 根拠なき断定 | リピートしたという購入者もいます(出典明記) |
| 業界最安値 | 根拠なき有利性の強調 | セール時の価格は商品ページをご確認ください |
| 今だけ半額(常時実施) | 期間限定を装う有利誤認 | 現在のセール価格は販売ページでご確認ください |

## ステマ規制: PR表記の必須

### 概要

2023年10月1日から、事業者の表示であるのに一般消費者が判別困難な表示は景表法違反(ステマ)となりました。アフィリエイト広告も対象です。2026年時点でも消費者庁の執行は継続しているとされます(要確認)。

規制対象は表示内容の決定に関与した事業者(広告主)です。ただしアフィリエイト記事は、広告主が内容に関与する構造のため、PR表記が必要になる典型例とされます。書き手としてもPR表記を徹底するのが安全です。

### 表記のルール

消費者庁の運用基準では、「広告」「宣伝」「プロモーション」「PR」といった用語で事業者の表示だと明瞭に示す方法が挙げられています。

- 記事の冒頭など、消費者がすぐ気づく位置に置きます。
- 文字が小さい、色が薄い、大量のタグに埋もれる、といった見落としやすい表示はNGと判断されるおそれがあります。
- 判断基準は特定の一文ではなく、表示全体から消費者が受ける印象とされます。

### NG→OK言い換え(ステマ)

| NG例 | NGの理由 | OK例 |
| --- | --- | --- |
| 表記なしで商品を絶賛 | 広告と判別困難 | 冒頭に「本記事はアフィリエイト広告を含みます」 |
| 文末に薄い小文字で「PR」 | 見落としやすく判別困難 | 冒頭に明瞭な大きさで「PR」「広告」 |
| 大量のハッシュタグに #PR を紛れ込ませる | 埋没して判別困難 | 独立した位置に明示 |
| 「个人の感想」を装い広告と隠す | 第三者を装う表示 | 提供関係を明記したうえで感想を書く |

## 運用チェックリスト

記事公開前に次を確認します。

- 体への効能効果(治る・効く・改善・予防・安心・安全)を書いていないか。
- 気分・雰囲気・使用シーンの表現に置き換えているか。
- 根拠のない最上級(最強・絶対・No.1)を使っていないか。
- 価格・割引の見せ方が実態と合っているか。
- 打消し表示が読める大きさ・配置になっているか。
- 冒頭に明瞭なPR表記があるか。
- 判断に迷う表現は専門家に確認したか。
