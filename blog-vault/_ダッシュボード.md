# 同人音声ブログ ダッシュボード

このノートは **Dataview プラグイン** で記事ノートを自動集計します。
記事を増やしても手で更新する必要はありません（`記事/` 配下にノートを足すだけ）。

> [!info] 使うには
> 設定 → コミュニティプラグイン → **Dataview** をインストールして有効化してください。
> 有効化すると下のコードブロックが表が表に変わります。

---

## 優先度ランキング（全記事）

```dataview
TABLE WITHOUT ID
  link(file.link, ニッチ) AS "ニッチ",
  優先度スコア AS "優先度",
  ステータス AS "状態"
FROM "記事"
SORT 優先度スコア DESC, file.name ASC
```

## ステータス別カンバン

### アイデア
```dataview
LIST 優先度スコア
FROM "記事"
WHERE ステータス = "アイデア"
SORT 優先度スコア DESC
```

### 下書き中
```dataview
LIST 優先度スコア
FROM "記事"
WHERE ステータス = "下書き"
SORT 優先度スコア DESC
```

### 公開済み
```dataview
TABLE WITHOUT ID
  link(file.link, ニッチ) AS "ニッチ",
  公開URL AS "URL"
FROM "記事"
WHERE ステータス = "公開"
SORT file.name ASC
```

## 最有力候補（スコア16以上）

```dataview
TABLE WITHOUT ID
  link(file.link, ニッチ) AS "ニッチ",
  優先度スコア AS "優先度",
  差別化 AS "差別化点"
FROM "記事"
FLATTEN スコア.差別化 AS 差別化
WHERE 優先度スコア >= 16
SORT 優先度スコア DESC
```
